Theorem 9.3 — Detailed Proof

ガウス・マルコフの定理 Gauss–Markov Theorem

最小2乗推定量が最良線形不偏推定量（BLUE）であることの証明を、行間を補いながら丁寧に解説します。

ガウス・マルコフの定理

線形回帰モデル $\bm{y}=\bm{X}\bm{\beta}+\bm{u}$ において、 $E[\bm{u}]=\bm{0}$, $\Cov(\bm{u})=\sigma^2\bm{I}$ を仮定する。
このとき、最小2乗推定量 $\hat{\bm{\beta}}=(\bm{X}\T\bm{X})^{-1}\bm{X}\T\bm{y}$ は、 線形かつ不偏な推定量の中で共分散行列を最小にする。
すなわち $\hat{\bm{\beta}}$ は 最良線形不偏推定量（BLUE: Best Linear Unbiased Estimator）である。

モデルの前提

$\bm{y}$ は $n\times 1$ の観測ベクトル、 $\bm{X}$ は $n\times(k+1)$ のフルランク計画行列、 $\bm{\beta}$ は $(k+1)\times 1$ の未知パラメータ、 $\bm{u}$ は $n\times 1$ の誤差ベクトルで、 $E[\bm{u}]=\bm{0}$, $E[\bm{u}\bm{u}\T]=\sigma^2\bm{I}$ とする。

証 明

1
一般の線形不偏推定量の設定

$\bm{\beta}$ の任意の線形推定量は、ある $(k+1)\times n$ 行列 $\bm{C}$ を用いて

$$\tilde{\bm{\beta}} = \bm{C}\bm{y}$$

と表される。この推定量が不偏であるための条件を求める。$\bm{y}=\bm{X}\bm{\beta}+\bm{u}$ を代入すると、

$$E[\bm{C}\bm{y}] = E[\bm{C}(\bm{X}\bm{\beta}+\bm{u})] = \bm{C}\bm{X}\bm{\beta} + \bm{C}\cdot\underbrace{E[\bm{u}]}_{=\bm{0}} = \bm{C}\bm{X}\bm{\beta}$$

これが任意の $\bm{\beta}$ に対して $E[\bm{Cy}]=\bm{\beta}$ を満たすためには、

$$\bm{C}\bm{X} = \bm{I}_{k+1}$$

が必要十分条件である。これが不偏性の条件である。

2
線形不偏推定量の共分散行列

不偏条件 $\bm{CX}=\bm{I}$ のもとで、$\bm{Cy}$ の共分散行列を計算する。

$$\Cov(\bm{Cy}) = E\bigl[(\bm{Cy}-\bm{\beta})(\bm{Cy}-\bm{\beta})\T\bigr]$$

ここで $\bm{Cy}-\bm{\beta}$ を整理する。

$$\bm{Cy}-\bm{\beta} = \bm{C}(\bm{X\beta}+\bm{u})-\bm{\beta} = \underbrace{\bm{CX}}_{=\bm{I}}\bm{\beta}+\bm{Cu}-\bm{\beta} = \bm{Cu}$$

不偏条件 $\bm{CX}=\bm{I}$ が効いて、$\bm{CX\beta}$ と $\bm{\beta}$ がぴったり相殺する。推定誤差は純粋に $\bm{Cu}$（誤差 $\bm{u}$ の線形変換）だけになる。

したがって、

$$\Cov(\bm{Cy}) = E[\bm{Cu}(\bm{Cu})\T] = E[\bm{C}\bm{u}\bm{u}\T\bm{C}\T] = \bm{C}\underbrace{E[\bm{u}\bm{u}\T]}_{=\sigma^2\bm{I}}\bm{C}\T = \sigma^2\bm{C}\bm{C}\T$$
3
OLS推定量との差を導入して $\bm{CC}\T$ を展開

最小2乗推定量（OLS）に対応する行列を

$$\bm{C}^* = (\bm{X}\T\bm{X})^{-1}\bm{X}\T$$

とおく。任意の線形不偏推定量の行列 $\bm{C}$ に対して、差

$$\bm{D} \equiv \bm{C}-\bm{C}^*$$

を定義する。すなわち $\bm{C}=\bm{D}+\bm{C}^*$ である。これを $\bm{CC}\T$ に代入して展開すると、

$$\bm{CC}\T = (\bm{D}+\bm{C}^*)(\bm{D}+\bm{C}^*)\T = \bm{D}\bm{D}\T + \bm{D}(\bm{C}^*)\T + \bm{C}^*\bm{D}\T + \bm{C}^*(\bm{C}^*)\T$$
4
交差項がゼロ — 証明の核心

$\bm{D}(\bm{C}^*)\T = \bm{0}$ を示す。これが証明の最も重要なステップである。

（4a）$\bm{DX}=\bm{0}$ の確認

$$\bm{DX} = (\bm{C}-\bm{C}^*)\bm{X} = \bm{CX} - \bm{C}^*\bm{X}$$

ここで、

$$\bm{CX} = \bm{I} \quad\text{（不偏条件）}$$ $$\bm{C}^*\bm{X} = (\bm{X}\T\bm{X})^{-1}\bm{X}\T\bm{X} = \bm{I}$$

よって、

$$\bm{DX} = \bm{I}-\bm{I} = \bm{0}$$

（4b）$(\bm{C}^*)\T$ の具体形

$\bm{C}^* = (\bm{X}\T\bm{X})^{-1}\bm{X}\T$ の転置を計算する。 転置の公式 $(\bm{AB})\T = \bm{B}\T\bm{A}\T$ を繰り返し適用すると、

$$(\bm{C}^*)\T = \bigl[(\bm{X}\T\bm{X})^{-1}\bm{X}\T\bigr]\T = (\bm{X}\T)\T\bigl[(\bm{X}\T\bm{X})^{-1}\bigr]\T = \bm{X}\bigl[(\bm{X}\T\bm{X})\T\bigr]^{-1}$$

逆行列の転置に関する注意： 一般に $(\bm{A}^{-1})\T = (\bm{A}\T)^{-1}$ が成り立つ。 証明の写真では $(\bm{X}\T\bm{X})^{-\top}$ という略記が使われているが、 これは $\bigl[(\bm{X}\T\bm{X})^{-1}\bigr]\T = \bigl[(\bm{X}\T\bm{X})\T\bigr]^{-1}$ の意味である。

ここで $\bm{X}\T\bm{X}$ は対称行列である（$(\bm{X}\T\bm{X})\T = \bm{X}\T(\bm{X}\T)\T = \bm{X}\T\bm{X}$）。 したがってその逆行列も対称であり、

$$\bigl[(\bm{X}\T\bm{X})\T\bigr]^{-1} = (\bm{X}\T\bm{X})^{-1}$$

よって、

$$(\bm{C}^*)\T = \bm{X}(\bm{X}\T\bm{X})^{-1}$$

（4c）交差項の計算

$$\bm{D}(\bm{C}^*)\T = \bm{D}\cdot\bm{X}(\bm{X}\T\bm{X})^{-1} = \underbrace{(\bm{DX})}_{=\bm{0}}(\bm{X}\T\bm{X})^{-1} = \bm{0}$$

また、その転置 $\bm{C}^*\bm{D}\T = \bigl[\bm{D}(\bm{C}^*)\T\bigr]\T = \bm{0}\T = \bm{0}$ も成り立つ。

$(\bm{C}^*)\T$ が $\bm{X}(\cdots)$ の形をしているので、左から $\bm{D}$ をかけると $\bm{DX}=\bm{0}$ が効いて全体がゼロになる。 これは OLS 推定量の「直交性」とも呼ばれる性質である。

5
不等式の導出

交差項が消えたので、$\bm{CC}\T$ は次のように簡潔に書ける：

$$\bm{CC}\T = \bm{C}^*(\bm{C}^*)\T + \bm{DD}\T$$

$\sigma^2$ をかけると共分散行列の関係式が得られる：

$$\Cov(\bm{Cy}) = \sigma^2\bm{C}^*(\bm{C}^*)\T + \sigma^2\bm{DD}\T = \Cov(\hat{\bm{\beta}}) + \sigma^2\bm{DD}\T$$

ここで $\bm{DD}\T$ が非負定値（半正定値）行列であることを確認する。任意のベクトル $\bm{a}\in\mathbb{R}^{k+1}$ に対して、

$$\bm{a}\T\bm{DD}\T\bm{a} = (\bm{D}\T\bm{a})\T(\bm{D}\T\bm{a}) = \|\bm{D}\T\bm{a}\|^2 \geq 0$$

したがって $\sigma^2\bm{DD}\T \geq \bm{0}$（半正定値の意味で）であり、

$$\Cov(\bm{Cy}) \geq \Cov(\hat{\bm{\beta}})$$

ここで行列の不等式 $\bm{A}\geq\bm{B}$ は「$\bm{A}-\bm{B}$ が非負定値行列」であることを意味する。

「共分散行列が小さい」の意味： $\Cov(\bm{Cy})-\Cov(\hat{\bm{\beta}})\geq\bm{0}$ は、 任意の方向 $\bm{a}$ について $\bm{a}\T\Cov(\bm{Cy})\bm{a}\geq\bm{a}\T\Cov(\hat{\bm{\beta}})\bm{a}$ を意味する。特に $\bm{a}=\bm{e}_j$（第 $j$ 成分が1の単位ベクトル）をとれば、 個々のパラメータ $\beta_j$ の分散についても OLS が最小であることがわかる。

以上より、任意の線形不偏推定量 $\bm{Cy}$ の共分散行列は、 OLS推定量 $\hat{\bm{\beta}}$ の共分散行列以上（半正定値の意味で）となる。

すなわち $\hat{\bm{\beta}}=(\bm{X}\T\bm{X})^{-1}\bm{X}\T\bm{y}$ は 共分散行列を最小にする線形不偏推定量、 つまり最良線形不偏推定量（BLUE）である。

$\blacksquare$

直観的な理解

OLS以外の線形不偏推定量を使うと、差分 $\bm{D}=\bm{C}-\bm{C}^*$ に由来する $\sigma^2\bm{DD}\T$ の分だけ「余計な分散」が共分散行列に上乗せされる。 $\bm{DD}\T$ は常に非負定値なので、 OLS推定量の分散を下回ることは原理的に不可能である。 言い換えれば、OLS推定量は「無駄のない」推定量であり、 制約 $\bm{CX}=\bm{I}$（不偏性）を満たす範囲で 最も効率的に情報を抽出している。
